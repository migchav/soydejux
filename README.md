# SoydeJux 🏺
## Cultura, Costumbres y Tradiciones de la Mixteca

Portal cultural dedicado a preservar y difundir la herencia de la región Mixteca de Oaxaca, México.

---

## 🚀 Despliegue en GitHub Pages

1. **Fork o clona** este repositorio
2. Renombra el repositorio a `soydeJux` (o el nombre que prefieras)
3. Ve a **Settings → Pages**
4. En **Source**, selecciona `Deploy from a branch`
5. Selecciona la rama `main` y la carpeta `/ (root)`
6. Haz clic en **Save**
7. ⚠️ **Actualiza las URLs**: Reemplaza `yourusername` con tu usuario de GitHub en:
   - `index.html` (meta tags canonical, og:url)
   - `pages/*.html` (meta tags canonical)
   - `sitemap.xml` (todas las URLs)
   - `robots.txt` (Host y Sitemap)
   - `_config.yml` (url y baseurl)

Tu sitio estará disponible en: `https://yourusername.github.io/soydeJux/`

## 📁 Estructura del Proyecto

```
soydeJux/
├── index.html              # Página principal
├── css/
│   └── style.css           # Estilos del sitio
├── js/
│   └── main.js             # JavaScript (menú, galería, búsqueda)
├── pages/
│   ├── historia.html       # Historia de la Mixteca
│   ├── tradiciones.html    # Tradiciones y Festividades
│   ├── gastronomia.html    # Gastronomía Mixteca
│   ├── artesanias.html     # Arte y Artesanías
│   ├── lengua.html         # Lengua Mixteca
│   ├── municipios.html     # Municipios y Comunidades
│   ├── musica.html         # Música Tradicional
│   ├── vestimenta.html     # Vestimenta Típica
│   ├── galeria.html        # Galería Fotográfica
│   └── sitemap.html        # Mapa del Sitio (HTML)
├── robots.txt              # Configuración para motores de búsqueda
├── sitemap.xml             # Sitemap XML para SEO
└── _config.yml             # Configuración GitHub Pages
```

## ✨ Características

- **Menú dinámico** con dropdowns expandibles y versión móvil
- **Campo de búsqueda** en el encabezado con overlay de resultados
- **Mini-galerías** de 3 imágenes con controles y auto-avance en cada página
- **Footer completo** con créditos, redes sociales y enlaces
- **Iconos de redes sociales**: Facebook, YouTube, Instagram, TikTok
- **SEO optimizado**: robots.txt, sitemap.xml, meta tags completos, JSON-LD
- **Diseño responsivo** para móvil, tablet y escritorio
- **10 páginas completas** con contenido sobre cultura mixteca
- **Colores neutros** con tonos tierra inspirados en la Mixteca
